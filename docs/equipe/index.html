Equipe

Essa iniciativa é feita de forma voluntária e para fins educacionais pelo Observatório Covid-19 BR.

Temos o apoio das iniciativas Horas da Vida, Grupo Infovid, Rede Análise COVID-19 e Todos pelas Vacinas.

Este site foi desenvolvido com a ferramenta Hugo, utilizando o tema Learn, que por sua vez é baseado no Grav.
